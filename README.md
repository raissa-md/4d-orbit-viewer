
# 4D Orbit Viewer

Visualize spacecraft positions and orbits in space and time.

## Description

The 4D Orbit Viewer is an interactive web application designed for the high-fidelity visualization of spacecraft trajectories and planetary positions across both space and time. By integrating data from the Satellite Situation Center (SSC) database, the tool provides a dynamic 4D environment for researchers to analyze complex orbital mechanics and mission geometries.

**Key Features**

* Extensive Spacecraft Catalog: Real-time and historical orbit data for over 200 spacecraft, allowing for comparative mission analysis.

* Virtual 3D Environment: A full-scale spatial model featuring the Sun, the Moon, and the four inner planets (Mercury, Venus, Earth, and Mars) and Lagrange Point 1 (L1).

* Space Physics Visualization: Includes visual representations of the bowshock and the magnetopause.

* Flexible Reference Frames: Support for a variety of geocentric and heliocentric coordinate systems, enabling users to toggle between different physical perspectives depending on their research needs.
